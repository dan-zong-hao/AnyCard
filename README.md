# Time Magic (时间魔术) 🎩✨

这是一个伪装成手机“息屏显示 (Always-On Display)”的数字魔术 App。魔术师可以通过隐秘的操作，利用时间数字和屏幕区域，精准预言或变出观众任意选择的扑克牌。

## ✨ 主要功能

*   **拟真界面**：极简的黑色背景，模拟真实的手机待机画面（时间、日期、电量、通知图标）。
*   **隐形操作**：屏幕被隐形划分为不同区域，配合时间逻辑触发魔术。
*   **数字牌系统 (A-10)**：基于秒数自动计算点数，无需记忆。
*   **人头牌系统 (J/Q/K)**：通过点击时间数字进行“预设锁定”。
*   **大小王系统**：特殊的隐藏触发区域。
*   **震动反馈**：操作成功时会有微弱震动提示（需设备支持）。

## 📖 魔术师手册 (操作指南)

### 1. 花色选择 (屏幕四象限)

屏幕被隐形划分为四个区域，点击不同区域决定弹出的花色：

| 区域 | 花色 | 记忆口诀 |
| :--- | :--- | :--- |
| **左上** | ♠️ 黑桃 (Spades) | - |
| **右上** | ♥️ 红心 (Hearts) | - |
| **左下** | ♣️ 梅花 (Clubs) | - |
| **右下** | ♦️ 方片 (Diamonds) | - |

---

### 2. 数字牌变法 (A - 10)

**逻辑：尾数法**
系统会自动读取你点击屏幕那一瞬间的**秒数最后一位**。

*   **1** = A
*   **2 ~ 9** = 数字 2 ~ 9
*   **0** = 10

**表演流程：**
1.  观众说：“我要一张红心 7”。
2.  你假装感应时间，余光观察秒针。
3.  等到秒数变为 `17`, `27`, `37`, `47` 或 `57` 的瞬间。
4.  点击屏幕**右上角**（红心区域）。
5.  屏幕弹出红心 7。

---

### 3. 人头牌变法 (J, Q, K)

**逻辑：预设锁定法**
为了随时变出 JQK，而不需要等待特定的秒数，你需要先点击时间数字进行“锁定”。

*   点击 **时 (Hours)** → 下一次点击锁定为 **J**
*   点击 **分 (Minutes)** → 下一次点击锁定为 **Q**
*   点击 **秒 (Seconds)** → 下一次点击锁定为 **K**

**表演流程：**
1.  观众说：“黑桃 K！”
2.  你说：“让我把时间定格在 K 的时刻……”（手指假装抚摸时间，实际快速点击一下**秒数**的位置）。
3.  此时系统已锁定为 K。
4.  你说：“就在这里！”然后点击**左上角**（黑桃区域）。
5.  屏幕弹出黑桃 K。

---

### 4. 大小王 (Jokers)

特殊的隐藏触发点，无需配合花色区域：

*   **大王 (Big Joker)**：点击屏幕中间的 **日期文本** (例如: 26/01/18 周日)。
*   **小王 (Small Joker)**：点击屏幕下方的 **通知图标栏** (电话/短信/邮件图标)。

## 🛠️ 技术栈

*   **React 19**
*   **TypeScript**
*   **Tailwind CSS** (用于布局和动画)
*   **Vibration API** (用于触感反馈)

## 🚀 安装与运行

1.  克隆仓库
2.  安装依赖：
    ```bash
    npm install
    ```
3.  启动项目：
    ```bash
    npm start
    ```

## 📱 最佳体验设置

*   **全屏模式**：点击屏幕右下角极小的隐形区域可切换全屏，隐藏浏览器地址栏。
*   **手机设置**：建议将手机亮度调高，并关闭自动锁屏，以获得最佳的表演效果。
